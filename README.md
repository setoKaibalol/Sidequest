# SideQuest

SideQuest is an external opt-in web application that helps Reddit users discover relevant opportunities outside Reddit.

Users may connect their own Reddit account through OAuth and voluntarily select individual public posts or comments as evidence of their interests, experience, or knowledge.

## Planned Reddit API use

- OAuth account verification
- Read-only access to the authenticated user's own public content
- User-controlled selection of individual posts and comments
- Clear attribution and links to original Reddit content

SideQuest will not post, vote, message users, monitor subreddits, scrape Reddit, sell Reddit data, or use Reddit content for AI training.

Status: Early development / API approval pending.
